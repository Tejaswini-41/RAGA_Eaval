## Summary

The current PR involves refactoring the code to replace `attachPrompts` with `attachPrompt` and related changes. The key changes include:

*   Replacing `attachPrompts` with `attachPrompt` to handle attaching a single prompt to a chat input.
*   Introducing `detachPrompt` to detach a prompt from a chat input.
*   Updating the return type of `attachPrompt` to include a boolean indicating whether the prompt was already attached.

The purpose of these changes is to improve the handling of chat prompts and attachments.
