## Summary

The current PR involves refactoring the code to replace the `attachPrompts` function with `attachPrompt` and introducing a new `detachPrompt` function. The purpose of these changes is to modify the attachment and detachment of prompts to a chat input.

Key changes:

* Replaced `attachPrompts` with `attachPrompt` to attach a single prompt to a chat input.
* Introduced `detachPrompt` to detach a prompt from a chat input.
* Modified the return type of `attachPrompt` to include a boolean indicating whether the prompt was already attached.
